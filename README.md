# todo-application
Todo Application Using:
* Bullet VueJS 3
* Bullet VueX
* Bullet Bootstrap
* Bullet indexDB
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
